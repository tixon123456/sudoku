\# Игра Судоку на Python с Tkinter

\## Описание

Игра "Судоку" с графическим интерфейсом на Python, реализованная с использованием Tkinter. 

Программа генерирует случайные головоломки трех уровней сложности, проверяет корректность 

ввода и предоставляет возможность проверки решения.


\## Архитектура

sudoku-game/

├── src/

│   ├── main.py          # Основной файл с GUI и игровой логикой

│   └── init.py      # Инициализационный файл пакета

├── Dockerfile           # Конфигурация Docker

├── requirements.txt     # Зависимости Python

├── .gitignore          # Игнорируемые файлы Git

└── README.md           # Документация


\## Функциональность

\- Генерация случайных головоломок судоку

\- Три уровня сложности: Легкая, Средняя, Сложная

\- Автоматическая проверка корректности ввода

\- Подсветка ошибочных ячеек

\- Проверка полного решения

\- Защита начальных цифр от изменения
